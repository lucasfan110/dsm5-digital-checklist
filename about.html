About This Website

Made by Lucas Fan

The purpose of this project is to make DSM-5 more accessible to everyone. By providing an online interactive checklist, and an easy way to search through and browse through each disorder, I hope to create a tool that allows people to have easier access to the diagnostic criteria used by many clinicians around the US. It can also be used by students to study more in depth about how each disorder is described in the DSM, which help serve as an official dictionary for each disorder

This project was created through first using a Python script to scrap the data from DSM-5. Through this process, it was able to generate 15 JSON files, each containing 11 diagnostic criteria, and the bullet points that go into each illness that provides a diagnosis. Through these 15 JSON files, I was able to generate an HTML page for each diagnostic criteria based on the illness name and all the criteria that goes along with it in the JSON file. Through this process, I was able to generate a digital checklist for ~160 disorders that are listed on the DSM-5. Then, I categorized each disorder based on the broader category used in the DSM-5, to provide easy access and search through for each disorder on the home page. The website also contains a search feature which allows the user to search through all the disorders listed on the website.

Each digital checklist, there is a checkbox next to each bullet point, which the user can click once a patient satisfies a criteria. To be diagnosed, all criteria must be checked and fulfilled. There are also special conditions, such as specifying if a patient has a particular condition, as well as extra notes, which are also listed on the digital checklist. The user can type into the textbox next to each “specify if” and such to provide more detailed information towards a patient’s diagnosis.
