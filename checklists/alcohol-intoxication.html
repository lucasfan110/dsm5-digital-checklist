Alcohol Intoxication

  1. A. Recent ingestion of alcohol.
  2. B. Clinically significant problematic behavioral or psychological changes (e.g., inappropriate sexual or aggressive behavior, mood lability, impaired judgment) that developed during, or shortly after, alcohol ingestion.
  3. C. One (or more) of the following signs or symptoms developing during, or shortly after, alcohol use:
    1. 1. Slurred speech.
    2. 2. Incoordination.
    3. 3. Unsteady gait.
    4. 4. Nystagmus.
    5. 5. Impairment in attention or memory.
    6. 6. Stupor or coma.
  4. D. The signs or symptoms are not attributable to another medical condition and are not better explained by another mental disorder, including intoxication with another substance.
  5. Coding note:

    The ICD-10-CM code depends on whether there is a comorbid alcohol use disorder. If a mild alcohol use disorder is comorbid, the ICD-10-CM code is F10.120, and if a moderate or severe alcohol use disorder is comorbid, the ICD-10-CM code is F10.220. If there is no comorbid alcohol use disorder, then the ICD-10-CM code is F10.920. 562
