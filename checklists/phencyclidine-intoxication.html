Phencyclidine Intoxication

  1. A. Recent use of phencyclidine (or a pharmacologically similar substance).
  2. B. Clinically significant problematic behavioral changes (e.g., belligerence, assaultiveness, impulsiveness, unpredictability, psychomotor agitation, impaired judgment) that developed during, or shortly after, phencyclidine use. 595

  4. Note:

    When the drug is smoked, “snorted,” or used intravenously, the onset may be particularly rapid.

  5. D. The signs or symptoms are not attributable to another medical condition and are not better explained by another mental disorder, including intoxication with another substance.
  6. Coding note:

    The ICD-10-CM code depends on whether there is a comorbid phencyclidine use disorder. If a mild phencyclidine use disorder is comorbid, the ICD-10-CM code is F16.120, and if a moderate or severe phencyclidine use disorder is comorbid, the ICD-10-CM code is F16.220. If there is no comorbid phencyclidine use disorder, then the ICD-10-CM code is F16.920.
