Sedative, Hypnotic, or Anxiolytic Intoxication

  1. A. Recent use of a sedative, hypnotic, or anxiolytic.
  2. B. Clinically significant maladaptive behavioral or psychological changes (e.g., inappropriate sexual or aggressive behavior, mood lability, impaired judgment) that developed during, or shortly after, sedative, hypnotic, or anxiolytic use. 627
  3. C. One (or more) of the following signs or symptoms developing during, or shortly after, sedative, hypnotic, or anxiolytic use:
    1. 1. Slurred speech.
    2. 2. Incoordination.
    3. 3. Unsteady gait.
    4. 4. Nystagmus.
    5. 5. Impairment in cognition (e.g., attention, memory).
    6. 6. Stupor or coma.
  4. D. The signs or symptoms are not attributable to another medical condition and are not better explained by another mental disorder, including intoxication with another substance.
  5. Coding note:

    The ICD-10-CM code depends on whether there is a comorbid sedative, hypnotic, or anxiolytic use disorder. If a mild sedative, hypnotic, or anxiolytic use disorder is comorbid, the ICD-10-CM code is F13.120, and if a moderate or severe sedative, hypnotic, or anxiolytic use disorder is comorbid, the ICD-10-CM code is F13.220. If there is no comorbid sedative, hypnotic, or anxiolytic use disorder, then the ICD-10-CM code is F13.920.
